# StyleViewer.js
用於查看網頁中元素樣式的 JavaScript 庫

## 使用方式
```html
<!-- 在網頁中加入以下代碼 -->
<script src="https://siyu1017.github.io/StyleViewer.js/dist/StyleViewer.min.js"></script>
```

詳情 : https://siyu1017.github.io/StyleViewer.js/

💡單擊 Alt 以切換 Popup 是否移動

## 更新內容
- 符合的 Selector 高亮
- 辨別 attribute 受否被瀏覽器支持
- 辨別 value 是否合法
- 辨別 variable 是否已定義
